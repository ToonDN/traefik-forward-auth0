Prerequisites
--------------
- Registered Auth0 account.
- Deployed Traefik in Kubernetes
- Traefik 2.X Custom Resource Definitions, CRDs, installed in Kubernetes
- Traefik 2.X KubernetesCRD-provider enabled in Traefik.
